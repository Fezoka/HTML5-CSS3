Principais formatações

Negrito / Destaque

Nessa frase, usamos a tag <b> cu </b> para deixar em Negrito (não semântica).

E nessa, temos uma <strong> strong </strong> (semântica).

Itáilico / Ênfase

Nesta frase, temos um termo <i> itálico </i> (não semântica).

Nesta frase, temos um termo em <em> Ênfase </em> (semântica).

Texto marcado

Podemos criar também um <mark> texto marcado ROSA </mark>.

E nesse temos a cor <mark> Padrão do site </mark> (Padrão de marcação).

Texto grande e pequeno

Estamos criando um <big> texto Grande </big> e um <small> texto Pequeno </small> neste parágrafo.

Texto deletado

Podemos marcar um <del> texto como excluido ></del> para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um <ins> texto como inserido </ins> para dar um ênfase e indicar que ele foi adicionado depois.

Existe também o <u> sublinhado </u> que não é semântico.

Texto sobrescrito

Para inserir coisas do tipo X2+3, com a tag <sup> </sup>

Texto subscrito

Para inserir coisas do tipo H2O, com a tag <sub> </sub>
